# 電卓UIモックアップ

以下は tkinter 実装を想定したシンプルな外見モックアップです。

```text
┌──────────────────────────────┐
│ Calculator                   │
├──────────────────────────────┤
│                    12+7×3    │  ← 式表示（任意）
│                         33   │  ← 結果表示
├────────┬────────┬────────┬───┤
│   AC   │   C    │   ⌫    │ ÷ │
├────────┼────────┼────────┼───┤
│   7    │   8    │   9    │ × │
├────────┼────────┼────────┼───┤
│   4    │   5    │   6    │ - │
├────────┼────────┼────────┼───┤
│   1    │   2    │   3    │ + │
├────────┼────────┼────────┼───┤
│  +/-   │   0    │   .    │ = │
└────────┴────────┴────────┴───┘
```

## デザイン方針
- 結果表示は右寄せ
- 演算子列は視認性のため背景色を変える（例: 薄いオレンジ）
- `=` ボタンは強調色を使用
- 各ボタンは同サイズのグリッド配置で、クリックしやすさを優先
